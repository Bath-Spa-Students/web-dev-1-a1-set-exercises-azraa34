HTML TAGS

<h1>

Main heading, used for important titles or sections.

<h1>Main Heading</h1>

<p>

Paragraph tag, used for text content.

<p>This is a paragraph.</p>

<br>

Line break tag, used to break lines within a paragraph.

<p>First line<br>Second line</p>

<ul> & <ol>

Unordered and ordered list tags, used to create lists.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<ol>
    <li>Step 1</li>
    <li>Step 2</li>
</ol>

<li>

List item tag, used within ul or ol to define list items.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<div>

Division tag, used for grouping and styling elements.

<div class="container">
    <p>Content inside the div.</p>
</div>

<i>

Italic tag, used to italicize text.

<p>This is <i>italic</i> text.</p>

<a>

Anchor tag, used to create hyperlinks.

<a href="https://example.com">Link Text</a>

<b>

Bold tag, used to make text bold.

<p>This is <b>bold</b> text.</p>

<img>

Image tag, used to display images.

<img src="image.jpg" alt="Description of image">

CSS PROPERTIES

font-family

Specifies the font family for text content within an element.

body {
    font-family: 'Arial', sans-serif;
}

color

Sets the text color of an element's content.

p {
    color: #333;
}

background-color

Defines the background color of an element.

body {
    background-color: #f7f7f7;
}

padding

Adds space inside an element's border.

article {
    padding: 10px;
}

margin

Sets the margin space outside an element's border.

section {
    margin: 20px;
}

text-align

Aligns the text content within an element.

h1 {
    text-align: center;
}

font-weight

Specifies the weight (thickness) of the font.

h2 {
    font-weight: bold;
}

border

Sets the border properties of an element.

div {
    border: 1px solid #ccc;
}

border-radius

Defines the curvature of an element's border corners.

button {
    border-radius: 5px;
}

box-shadow

Adds a shadow effect to an element's box.

div {
    box-shadow: 2px 2px 5px rgba(0, 0, 0, 0.3);
}

text-decoration

Adds decoration to text, such as underline or strikethrough.

a {
    text-decoration: none;
}

a:hover {
    text-decoration: underline;
}
